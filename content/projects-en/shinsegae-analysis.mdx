## Shinsegae Department Store Food Hall Analysis & Full-Floor Expansion Frame Design

## Project Overview

Shinsegae Department Store had diverse analysis requests for the food hall and full-floor spaces, but hypotheses were mixed without criteria for what to verify first or how to expand.
Structured 73 client requests by data analysis feasibility and business needs, then analyzed event effects using statistical testing, expanding to a vertical analysis frame applicable to all floors and movement clustering proposals.
Built not a one-off report but a stepwise expandable analysis roadmap, managing data integrity issues through face-to-face communication to maintain project continuity and client trust.

## Approach

### 1) Structured 73 Client Requests into an Analysis Roadmap

Client requests were numerous but differed in nature — which questions to answer first and what was feasible with current data wasn't organized.
- First classified 73 client requests by business needs and data analysis feasibility.
- Then organized into a matrix format showing which hypotheses each request connects to, whether short-term response is possible, and whether follow-up data design is needed.

Mixed client requests were converted from a one-off backlog into an analysis roadmap with priorities and expansion directions.

### 2) Converted Lunar New Year Event Effect into Verifiable Analysis

Client interest in whether visitors increased during the Lunar New Year event was high, but simple before/after comparison couldn't adequately explain actual event effects.
- Used the food hall as the first analysis target, separately analyzing when visitor changes began, whether genre differences existed, and how patterns compared to Chuseok.
- To go beyond simple increase/decrease reporting, set reference periods and applied 95% significance statistical testing to determine whether changes were meaningful.
- Also examined food genre differences to identify which categories showed prominent effects rather than overall increases.

Event analysis was converted from simple status summary into a hypothesis-validation report clients could actually use for decision-making.

### 3) Expanded Analysis to Full-Floor Vertical Frame

Even if food hall analysis value was confirmed, simply repeating it across all floors wouldn't sufficiently explain the department store's structural characteristics.
- Based on food hall analysis experience, judged that full-floor questions should center on inter-floor movement and linked behaviors rather than individual floor status.
- Designed a vertical analysis frame examining `floor roles`, `inter-floor movement`, and `purpose-visit patterns` at the full-floor level.
- Particularly, rather than dividing path types by fixed rules, applied K-Medoids clustering based on actual path patterns to typify shopping routes.

Established a full-floor expansion frame and follow-up analysis proposal structure capable of explaining the entire department store beyond individual analyses.

### 4) Managed Data Integrity Crisis Through Face-to-Face Communication

A data integrity issue discovered just before report delivery risked technical errors escalating directly into client trust decline.
- Treated this not as a simple fix request but as a project trust issue, choosing to meet the client directly to explain.
- Organized and transparently shared error causes and impact scope, explaining fix timeline and subsequent verification methods together.

Maintained a trust foundation enabling continued follow-up analysis and expansion discussions without project interruption or relationship deterioration.
